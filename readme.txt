!!!Prerequisites: Python (Python3 for Linux) version over 3.8 and Git
=====================================================================

1.Run the following commands:
 - on Windows:
 * pip -r requirements.txt (this will install the requirements for running the application)
 * flask --app app run (this will run a server at localhost on port 5000)
 - on Linux:
 * pip3 -r requirements.txt (this will install the requirements for running the application)
 * flask --app app run (this will run a server at localhost on port 5000)
2. Open the following link http://localhost:5000/ in your browser:
 - you can paste XML or JSON directly or you can upload a file. it will prompt you a download a converted file.
